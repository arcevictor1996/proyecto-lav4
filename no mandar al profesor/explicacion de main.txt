1. Importación de módulos
Estas líneas importan las funciones menu_alumnos, menu_instructores y menu_cursos desde los archivos correspondientes dentro de la carpeta src


Es importante destacar que el uso de from src.alumnos.menu_alumnos import menu_alumnos sugiere que el código está estructurado en módulos, lo que mejora la organización del proyecto.

2. Definición de la función menu_principal

La función menu_principal es el punto de entrada principal del sistema. Utiliza un bucle while True para que el menú se muestre repetidamente hasta que el usuario decida salir (opción "4").

3. Condiciones para manejar las opciones del menú

llama al la funcion que el usuario elije despues
Este bloque condicional verifica el valor de opcion y ejecuta una función correspondiente según la selección del usuario
¿Por qué se usa un bucle while True?

Se usa para que el menú principal se repita constantemente hasta que el usuario decida salir. El bucle continuará ejecutándose hasta que el usuario ingrese la opción "4", que termina el bucle con break.

4. Condición if __name__ == "__main__":
Esta línea es una condición especial que verifica si el archivo está siendo ejecutado como programa principal. Si es así, llama a la función menu_principal(), lo que significa que el menú principal se ejecutará al iniciar el script.

Esto garantiza que el código dentro de este bloque solo se ejecute cuando el archivo se ejecute directamente y no cuando se importe como módulo en otro archivo.
